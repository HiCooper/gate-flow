# 贡献指南

欢迎为 GateFlow 贡献代码和文档!

## 如何开始

1. Fork 本仓库
2. 创建特性分支 (`git checkout -b feature/your-feature`)
3. 提交更改 (`git commit -m 'feat: add your feature'`)
4. 推送到分支 (`git push origin feature/your-feature`)
5. 开启 Pull Request
