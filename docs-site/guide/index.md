# 产品指南

欢迎使用 GateFlow!本指南面向产品经理、运营人员和数据分析师,帮助你了解如何使用 GateFlow 平台进行 A/B 测试实验。

## 你将学到什么

- GateFlow 的核心能力和价值
- 如何创建和管理实验
- 如何阅读和理解实验报告
- 流量分配和协作的最佳实践
