# 流量地图

本文档介绍如何使用流量地图查看和管理实验流量分布。

## 功能概述

流量地图提供全局视角，展示所有实验的流量占用情况。

## 流量占用计算

流量占用 = 实验桶范围 × 实验状态

| 状态 | 实际流量 |
|------|----------|
| running | 100% 占用 |
| ramp | 按当前放量比例计算 |
| draft | 0%（未生效） |

## 冲突检测

### 同层冲突

同一层内实验的桶范围不能重叠：

```
Layer: default

实验 A: bucket 0-2999  ✓
实验 B: bucket 3000-5999  ✓
实验 C: bucket 2000-4999  ✗ 冲突！
```

创建实验时系统会自动检测冲突。

## 查看入口

管理控制台 -> 实验列表 -> 流量地图

## 优化建议

1. **避免长期占用大量流量**：实验结束后及时归档
2. **合理分层**：按业务域分离实验
3. **控制同时运行数量**：避免资源争抢
