# 统计显著性

本文档解释统计显著性的概念以及在 GateFlow 中的应用。

## 什么是统计显著性？

统计显著性用于判断实验结果的差异是否由随机因素导致。

在 A/B 测试中，我们假设：
- **H0 (零假设)**: 实验组与对照组无差异
- **H1 (备择假设)**: 实验组与对照组有差异

统计检验判断我们是否有足够证据拒绝 H0。

## p 值

p 值表示在零假设为真时，观察到当前或更极端结果的概率。

| p 值 | 结论 |
|------|------|
| p < 0.01 | 高度显著 |
| p < 0.05 | 统计显著 |
| p >= 0.05 | 不显著 |

## Z-Test

GateFlow 使用 Z-Test 进行显著性检验：

```
Z = (X_treatment - X_control) / sqrt(σ²_treatment/n_treatment + σ²_control/n_control)
```

- Z > 1.96 或 Z < -1.96: p < 0.05
- Z > 2.576 或 Z < -2.576: p < 0.01

## 置信区间

置信区间（Confidence Interval）表示效应量在一个置信水平下的可能范围。GateFlow 使用 **95% 置信区间**。

```
差异 d = p_treatment - p_control

SE = sqrt(p_c × (1-p_c) / n_c  +  p_t × (1-p_t) / n_t)

95% CI = [d - 1.96 × SE ,  d + 1.96 × SE]

提升 CI = CI / p_control
```

置信区间与 p-value 的关系：
- **CI 不包含 0** ↔ **p < 0.05**（统计显著）
- **CI 包含 0** ↔ **p ≥ 0.05**（不显著）

置信区间比 p-value 提供更多信息：

| 信息 | p-value | 置信区间 |
|------|---------|----------|
| 是否显著 | ✓ | ✓ |
| 效应大小 | 不直接 | ✓ |
| 估计精度 | 不直接 | ✓（区间宽度） |
| 最坏/最好情况 | 不直接 | ✓（上下限） |

## 示例

```
Control:   转化率 5.2%, 样本 10000
Treatment: 转化率 6.1%, 样本 10000

绝对差异 d = 6.1% - 5.2% = 0.9%
SE = sqrt(0.052×0.948/10000 + 0.061×0.939/10000) = 0.0033
95% CI = [0.9% - 1.96×0.33% , 0.9% + 1.96×0.33%]
       = [0.25% , 1.55%]
提升   = 0.9% / 5.2% = +17.3%
提升 CI = [4.8% , 29.8%]

Z = 0.9% / 0.0033 = 2.73
p = 0.006 < 0.05 → 差异显著
```

结论：实验组转化率提升 17.3%，95% 置信区间为 +4.8% ~ +29.8%。即使在最保守估计下，仍有 4.8% 的提升。

## 注意事项

1. **显著性 ≠ 业务价值**: 即使显著，也要看效果大小
2. **样本量影响**: 大样本下微小差异也显著
3. **避免 p-hacking**: 不要在看到结果后才决定检验方法
