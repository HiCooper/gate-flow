# 权限与协作

本文档介绍 GateFlow 的权限模型和团队协作功能。

## 角色定义

| 角色 | 权限说明 |
|------|----------|
| 管理员 | 完整权限，包括系统配置 |
| 实验Owner | 创建、编辑、启动、停止自己创建的实验 |
| 分析师 | 查看所有实验报告，无编辑权限 |
| 访客 | 查看自己参与的实验 |

## 权限矩阵

| 操作 | 管理员 | 实验Owner | 分析师 | 访客 |
|------|--------|-----------|--------|------|
| 创建实验 | ✓ | ✓ | - | - |
| 编辑实验 | ✓ | 仅Own | - | - |
| 启动/停止实验 | ✓ | 仅Own | - | - |
| 查看报告 | ✓ | ✓ | ✓ | - |
| 管理用户 | ✓ | - | - | - |

## 协作流程

### 实验审核

```
创建者提交审核 -> 审核人审批 -> 通过/驳回
```

### 团队成员

- 实验创建者自动成为 Owner
- Owner 可添加其他成员为协作者

## 白名单管理

白名单用于：
- 内部测试账号
- 指定用户强制分组
- 特殊用户排除
