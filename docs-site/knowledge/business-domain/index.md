# 业务领域知识

本文档沉淀 GateFlow 业务相关的知识和规则。

## 核心概念

### 实验 (Experiment)

在特定条件下对用户进行分组的测试活动。

### 变体 (Variant)

实验的不同版本，如对照组 (control) 和实验组 (treatment)。

### 层 (Layer)

流量隔离单元，同层层实验流量互斥，跨层流量正交。

### 桶 (Bucket)

分桶算法的基本单位，范围 0-9999。

## 分桶算法

GateFlow 使用 MurmurHash3 实现确定性分桶：

```
bucket = Hash(userId + "#" + layerId + "#" + salt) % 10000
```

## 设计理念

1. **确定性**: 相同输入始终产生相同输出
2. **均匀性**: 用户均匀分布在各变体
3. **可移植性**: 跨平台算法一致
4. **隔离性**: 分层设计避免实验干扰
