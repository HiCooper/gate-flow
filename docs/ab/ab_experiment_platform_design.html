各阶段详细设计

草稿 DRAFT
核心操作: 实验设计、流量配置、指标配置、版本配置
自动校验: 名称唯一性、流量比例合法性、指标注册检查、最小样本量预估
协作: 支持多人协作编辑，随时保存
审批 REVIEW
审批链: 直属Leader → 数据科学家 → 平台管理员
检查清单: 业务合理性、统计功效、流量冲突、护栏指标、伦理合规
结果: 全部通过→灰度 / 任一驳回→退回草稿
灰度 RAMP
三阶段: 1%(2h)→5%(4h)→10%(12h)
自动门禁: SRM检验、崩溃率、核心流程错误率
观察重点: 护栏指标趋势、主指标方向、样本均衡性
运行中 RUNNING
实时监控: 主/辅/护栏指标趋势看板
自动告警: 护栏恶化、SRM异常、样本不足
自动操作: 护栏p<0.01→暂停 / 样本达标→可分析通知
分析 ANALYZING
自动报告: 实验摘要、主/辅/护栏指标结果、时间趋势、人群拆分
智能建议: 统计显著且正向→全量 / 显著负向→回滚 / 不显著→延长或迭代
决策 DECISION
全量发布: 效果显著正向，推广全部用户（需管理层审批）
迭代优化: 有效果可优化，生成迭代草稿
回滚下线: 效果不显著或负向，恢复对照组
延长实验: 样本不足，需数据科学家确认

审批流配置

实验类型 审批链 说明
常规实验 直属Leader → 数据科学家 标准流程
高风险实验 直属Leader → 数据科学家 → 部门负责人 涉及核心流程或大额流量
