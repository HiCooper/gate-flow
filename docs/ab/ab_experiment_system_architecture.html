    03 分流模型 BUCKETING MODEL
    哈希分桶算法 · 域-层正交分流
    // 分桶公式
    bucket = MurmurHash3(user_id + layer_id) % 10000

    // 桶总数: 10000 → 粒度 0.1%
    // 拼接 layer_id → 层间正交
    // 同一用户 → 同一桶号(粘性)
