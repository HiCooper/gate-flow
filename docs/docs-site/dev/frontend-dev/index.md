# 前端开发

GateFlow 前端采用 Monorepo 架构,包含多个应用和共享组件库。

## 应用概览

| 应用 | 端口 | 说明 |
|------|------|------|
| admin | 3001 | 管理控制台 |
| marketing | 3000 | 营销站点 |
| victor-mng-front | 3002 | 实验管理 UI |

## 共享组件库

`packages/shared` 提供通用 UI 组件、Hooks、工具函数和设计令牌。
