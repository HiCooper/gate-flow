# 实验管理平台

实验管理平台是 AB 实验系统的 Web 控制台，连接分流 SDK、数据管道、统计分析引擎，提供从实验创建到决策归档的一站式管理能力。

## 平台定位

- **降低实验门槛**: 非技术用户也能自助创建、运行、分析实验
- **保障实验质量**: 内置 SRM 校验、统计功效计算、护栏监控等质量门禁
- **提升决策效率**: 自动化报告生成、智能决策建议
- **沉淀实验知识**: 实验知识库、经验复用

## 实验生命周期

```
[草稿 DRAFT] → [待审批 PENDING_APPROVAL] → [运行中 RUNNING]
                                                  │
                                                  ▼
                                           [已停止 STOPPED]
                                                  │
                                                  ▼
                                           [已归档 ARCHIVE]
```

### 各阶段说明

| 阶段 | 说明 | 关键操作 |
|------|------|---------|
| **草稿** | 实验设计阶段 | 填写假设、配置流量、选择指标、保存草稿 |
| **待审批** | 单步审批 | 审批通过则启动实验，驳回则退回草稿 |
| **运行中** | 正式实验期间 | 实时监控看板、自动告警；可启用 auto_ramp 灰度自动推进（1%→5%→10%→50%→100%），每阶段自动校验健康状态 |
| **已停止** | 数据收集结束 | 触发统计分析引擎生成报告（SRM → CUPED → Z-Test → mSPRT 护栏检查） |
| **已归档** | 知识沉淀 | 配置快照、数据保留、知识库收录 |

## 核心功能模块

### 实验管理

- **实验列表**: 多维度筛选（状态/时间/指标）、批量操作、快速克隆
- **实验详情**: 基本信息、流量配置、实时监控看板、统计报告 Tab 页
- **创建向导**: 分步表单（假设→流量→指标→版本），自动校验冲突

### 指标管理

- **指标注册中心**: 指标需注册审批，定义版本化管理
- **指标分层**:
  - **北极星指标**: 组织级核心指标（如人均 GMV、留存率）
  - **主指标**: 每个实验一个，衡量核心假设
  - **辅助指标**: 需 BH 多重校正
  - **护栏指标**: 全局监控，不可恶化（如崩溃率、页面加载 P90）

### 流量管理

- **流量地图**: 域/层可视化，直观展示各层实验流量占比
- **冲突检测**: 同层流量重叠检测、跨层正交校验、流量溢出告警

### 报表看板

| 报表 | 内容 | 频率 |
|------|------|------|
| 实时预览 | 运行中实验分钟级指标 | 5 分钟 |
| 日报 | 昨日全部实验指标 | 每日 |
| 周报 | 显著实验汇总 | 每周 |
| 月度复盘 | ROI、成功率、知识沉淀 | 每月 |

### 知识库

- 归档实验自动生成百科卡片（假设、结果、结论）
- 实验模板市场，快速克隆复用
- 经验标签标注，组织级实验文化积累

## 权限模型

### 角色定义

| 角色 | 说明 | 权限范围 |
|------|------|---------|
| ADMIN | 系统管理员 | 全部权限（含用户管理） |
| OPERATOR | 实验运营 | 创建/编辑/审批/查看实验和分析、流量查看、功效分析 |
| VIEWER | 只读用户 | 仅可查看实验和分析 |
| SDK_CLIENT | SDK 客户端 | 仅限分流/配置/事件 API（程序化访问） |

### 权限映射

| 权限 | 对应操作 |
|------|---------|
| `CREATE_EXPERIMENT` | 创建实验 |
| `EDIT_EXPERIMENT` | 编辑实验 |
| `DELETE_EXPERIMENT` | 删除实验 |
| `VIEW_EXPERIMENT` | 查看实验详情 |
| `APPROVE_EXPERIMENT` | 审批通过/驳回 |
| `SUBMIT_APPROVAL` | 提交审批 |
| `VIEW_TRAFFIC` | 查看流量地图 |
| `VIEW_ANALYSIS` | 查看统计分析报告 |
| `POWER_ANALYSIS` | 样本量/功效估算 |
| `MANAGE_USERS` | 用户及角色管理（仅 ADMIN） |

## 界面设计要点

- **信息密度**: 列表高密度展示，详情分层展开
- **状态可见**: 颜色+图标+文字三重编码实验状态
- **操作防错**: 关键操作二次确认
- **数据可视化**: 折线图(趋势)、柱状图(分组对比)、堆叠图(人群拆分)
