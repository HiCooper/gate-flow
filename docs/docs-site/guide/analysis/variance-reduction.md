# 方差缩减 (CUPED)

本文档介绍 CUPED 方差缩减技术。

## 什么是 CUPED?

CUPED (Controlled-experiment Using Pre-Experiment Data) 利用实验前数据降低指标方差，从而提高统计检验的灵敏度。

## 原理

```
Y_CUPED = Y - θ × (X - E[X])
```

- Y: 原始指标
- X: 实验前协变量（如历史转化率）
- θ: 回归系数，通过最小二乘法估计

## 效果

方差缩减程度取决于 Y 与 X 的相关性：

| 相关系数 | 方差缩减 |
|----------|----------|
| 0.1 | ~1% |
| 0.3 | ~9% |
| 0.5 | ~25% |
| 0.7 | ~49% |

## 应用场景

1. **指标波动大** - 减少随机噪声
2. **样本量有限** - 用更少样本达到显著
3. **效果较小** - 提高检验灵敏度

## 示例

假设：
- Control 组转化率: 5.0% ± 1.2%
- Treatment 组转化率: 5.3% ± 1.2%

使用 CUPED 后（假设历史相关性 0.5）：
- 有效方差减少 ~25%
- 检测到显著差异所需的样本量减少约 33%

## 与序贯检验对比

| 特性 | CUPED | 序贯检验 |
|------|-------|----------|
| 原理 | 方差缩减 | 连续监控 |
| 数据要求 | 需要历史数据 | 无 |
| 可叠加 | ✓ | ✓ |
