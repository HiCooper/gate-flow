# 阅读实验报告

本文档介绍如何阅读和理解 GateFlow 的实验报告。

## 报告结构

实验报告包含以下核心部分：

1. **实验概览** - 基本信息和运行状态
2. **主要指标对比** - 各变体指标数据
3. **统计显著性结果** - 显著性检验结果
4. **护栏指标监控** - 负面效应检测

## 关键指标解读

### 转化率

```
转化率 = 转化用户数 / 曝光用户数 × 100%
```

### 提升幅度

```
提升 = (treatment转化率 - control转化率) / control转化率 × 100%
```

## 报告示例

| 变体 | 样本量 | 转化率 | 提升 | p值 | 结论 |
|------|--------|--------|------|-----|------|
| control | 10,000 | 5.2% | - | - | 对照组 |
| treatment_a | 10,000 | 6.1% | +17.3% | 0.023 | 显著提升 ✓ |
| treatment_b | 10,000 | 4.8% | -7.7% | 0.156 | 无显著差异 |

## 结果判定

| 情况 | 说明 | 建议动作 |
|------|------|----------|
| p < 0.05 且提升为正 | 显著正向 | 可考虑全量 |
| p < 0.05 且提升为负 | 显著负向 | 建议停止 |
| p >= 0.05 | 无显著差异 | 继续观察或增加样本 |
