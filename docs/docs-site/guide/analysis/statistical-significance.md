# 统计显著性

本文档解释统计显著性的概念以及在 GateFlow 中的应用。

## 什么是统计显著性？

统计显著性用于判断实验结果的差异是否由随机因素导致。

在 A/B 测试中，我们假设：
- **H0 (零假设)**: 实验组与对照组无差异
- **H1 (备择假设)**: 实验组与对照组有差异

统计检验判断我们是否有足够证据拒绝 H0。

## p 值

p 值表示在零假设为真时，观察到当前或更极端结果的概率。

| p 值 | 结论 |
|------|------|
| p < 0.01 | 高度显著 |
| p < 0.05 | 统计显著 |
| p >= 0.05 | 不显著 |

## Z-Test

GateFlow 使用 Z-Test 进行显著性检验：

```
Z = (X_treatment - X_control) / sqrt(σ²_treatment/n_treatment + σ²_control/n_control)
```

- Z > 1.96 或 Z < -1.96: p < 0.05
- Z > 2.576 或 Z < -2.576: p < 0.01

## 示例

```
Control:   转化率 5.2%, 样本 10000
Treatment: 转化率 6.1%, 样本 10000

Z = (6.1% - 5.2%) / sqrt(0.052×0.948/10000 + 0.061×0.939/10000)
Z = 2.26

p = 0.024 < 0.05 → 差异显著
```

## 注意事项

1. **显著性 ≠ 业务价值**: 即使显著，也要看效果大小
2. **样本量影响**: 大样本下微小差异也显著
3. **避免 p-hacking**: 不要在看到结果后才决定检验方法
