# 架构决策记录 (ADR)

记录项目中的重要架构决策及其原因,便于后续理解和维护。

## 什么是 ADR?

ADR (Architecture Decision Record) 是一种轻量级的架构决策记录方式,包含:

- **标题**: 决策的简短描述
- **状态**: 提议/接受/废弃
- **背景**: 为什么要做这个决策
- **决策**: 我们决定怎么做
- **后果**: 决策带来的影响
