设计说明
1. expo-gateflow 为 Expo/React Native 跨平台 SDK，同时支持 iOS 和 Android
2. 使用 Zustand 进行状态管理，提供类似 Redux 的 API 但更轻量
3. Native Module 通过 expo-modules 暴露给 JS 层，实现跨平台调用
4. 支持事件驱动的编程模式，通过 addListener 订阅原生事件
5. 支持用户识别 (identify)、用户属性更新 (setUserAttributes)、实验评估 (evaluateExperiment)
6. 实验结果类型: matched (命中), holdout (未命中), noMatch (不匹配), notFound (未找到), error (错误)
